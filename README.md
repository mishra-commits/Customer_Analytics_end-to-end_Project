# Customer Analytics End-to-End Project

## Overview

This project analyzes customer behaviour, spending patterns, ratings, and subscription trends using Python, SQL, and Power BI.

## Tools Used

* Python
* SQL
* Power BI
* Power Query
* DAX

## Workflow

Raw Data → Python Cleaning → SQL Analysis → Power BI Dashboard

## Key Insights

* Customer Spending Analysis
* Subscription Trends
* Top & Bottom Rated Products
* Revenue Analysis

## Files Included

* Power BI Dashboard (.pbix)
* Python Notebook (.ipynb)
* SQL Queries (.sql)
* Dataset (.csv)
